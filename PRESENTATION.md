## 🎯 PROJECT OVERVIEW

### What is Scrunch & Create?

**Scrunch & Create** is a modern, fully responsive e-commerce platform specializing in handmade scrunchies and decorative hair accessories. The project demonstrates a complete end-to-end solution combining:

- ✅ **Web Development** - Full-stack e-commerce application
- ✅ **Business Analytics** - Cost breakdown and pricing strategies
- ✅ **Marketing Strategy** - Multi-channel presence and wholesale approach
- ✅ **DevOps** - Automated CI/CD pipeline with Docker and Jenkins
- ✅ **Customer Management** - Cart, authentication, and profile systems

### Project Goals

1. Create a user-friendly e-commerce platform
2. Develop sustainable manufacturing and pricing models
3. Establish wholesale partnerships
4. Build brand presence across social media
5. Implement automated deployment pipeline

## 💳 PRODUCT PRICING STRATEGY

### Wholesale Pricing Model

| Product | Cost Price (₹) | Wholesale (₹) | Profit (₹) | Profit % | MRP (₹) | Retailer Margin (₹) | Retailer % |
|---------|----------------|----------------|-----------|----------|--------|-----------------|------------|
| **Scrunchie** | 8.50 | 13.00 | 4.50 | 52.9% | 26 | 13.00 | 50.0% |
| **Tulip Scrunchie** | 14.35 | 21.00 | 6.65 | 46.3% | 42 | 21.00 | 50.0% |
| **Satin Hair Bow** | 16.50 | 24.00 | 7.50 | 45.5% | 48 | 24.00 | 50.0% |
| **Velvet Hair Bow** | 25.50 | 38.00 | 12.50 | 49.0% | 76 | 38.00 | 50.0% |
| **Sheer Satin Hair Bow** | 21.00 | 32.00 | 11.00 | 52.4% | 64 | 32.00 | 50.0% |
| **Satin Scarf Hairbow** | 22.00 | 32.00 | 10.00 | 45.5% | 64 | 32.00 | 50.0% |
| **Jimmi Choo (Chain)** | 24.08 | 36.00 | 11.92 | 49.5% | 72 | 36.00 | 50.0% |
| **Jimmi Choo (Pearls)** | 28.00 | 40.00 | 12.00 | 42.9% | 80 | 40.00 | 50.0% |

### Pricing Strategy Analysis

**Markup Strategy:**
- **Wholesale Margin:** 45-53% profit margin
- **Retail Strategy:** 2x markup on wholesale price (at MRP)
- **Retailer Incentive:** 50% margin for retail partners

**Price Tiers:**
```
Budget (₹8.50-14.35)     → Scrunchies
Standard (₹16.50-22)     → Hair Bows
Premium (₹24-28)         → Jimmi Choo Collection
Retail Price Range       → ₹26-₹80
```

**Key Insights:**
- ✅ Competitive wholesale pricing attracts retailers
- ✅ Standard 50% retailer margin incentivizes partnerships
- ✅ MRP at 2x wholesale ensures sustainable retail chain
- ✅ Higher profit margins on basic products (Scrunchies: 52.9%)
- ✅ Luxury items maintain reasonable margins (42.9-49.5%)

```
src/
├── componets/          # Reusable UI components
│   ├── banner/        # Hero section
│   ├── navbar/        # Navigation
│   ├── cart/          # Shopping cart logic
│   ├── productlist/   # Product display
│   └── filtersidebar/ # Product filters
├── pages/             # Page components
│   ├── home/         # Landing page
│   ├── products/     # Product listing
│   ├── cart/         # Shopping cart view
│   └── auth/         # Sign in/Sign up
└── assets/           # Images and static files
```

#### Key Implementation Features

**State Management:**
- React Context API for cart management
- Local storage persistence
- Real-time updates

**Routing:**
- Client-side routing with React Router
- Nested routes for pages
- Protected routes for auth pages

**Styling:**
- CSS Modules for component isolation
- Responsive design breakpoints
- Mobile-first approach

**Performance:**
- Vite for fast builds
- Code splitting
- Image optimization
- Lazy loading

### DevOps & CI/CD Pipeline

#### Pipeline Stages

```mermaid
GitHub Push
    ↓
Checkout Code
    ↓
Environment Setup
    ↓
Install Dependencies
    ↓
ESLint Code Quality
    ↓
Build React App (Vite)
    ↓
Build Docker Image
    ↓
Security Scan (Docker Scout)
    ↓
Push to Registry
    ↓
Deploy to Development
    ↓
Health Checks
    ↓
Deploy to Production (on tag)
    ↓
Success Notification
```

#### Docker Implementation

**Multi-stage Build:**
1. **Builder Stage** - Node 20 Alpine
   - Install dependencies
   - Build React application
   - Generate optimized bundle

2. **Production Stage** - Nginx Alpine
   - Copy built assets
   - Configure web server
   - Set security headers

**Image Optimization:**
- Base image: 166MB (Nginx Alpine)
- Final image size: ~50MB
- Build time: <2 minutes
- Startup time: <5 seconds

#### Jenkins Pipeline Features

```groovy
✅ GitHub webhook integration
✅ Automated testing
✅ Docker image scanning
✅ Registry push (Docker Hub)
✅ Development deployment
✅ Health verification
✅ Production deployment (tagged releases)
✅ Automated rollback capability
```

### Deployment Environments

#### Development
- Automatic deployment on git push to main
- Health checks every 30 seconds
- Container logging enabled
- Manual rollback option

#### Production
- Triggered on version tags (v*.*.*)
- Zero-downtime deployment
- Health verification (10-second grace period)
- Automatic rollback on health check failure
